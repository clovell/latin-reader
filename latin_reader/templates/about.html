ℹ️ About

Latin Reader is a web application that helps intermediate Latin students read prose and poetry by automatically parsing input text and displaying visual dependency trees.

It uses LatinCy (la_core_web_trf), a transformer-based Latin NLP model built on spaCy.

The treebank data is sourced from the Universal Dependencies project, specifically the Latin Perseus and UDante treebanks.
